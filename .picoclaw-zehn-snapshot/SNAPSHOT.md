# Picoclaw Zehn Runtime Snapshot

This is a sanitized documentation and metadata snapshot of
`/Users/aliai/.picoclaw-zehn`.

Included content is limited to operational context that is useful in source
control:

- agent identity and operating files
- memory markdown
- cron and state metadata
- operating prompts and reports
- audit documentation

The snapshot intentionally excludes runtime-sensitive and high-churn material:

- secrets and auth files
- launcher databases
- live config files
- session transcripts
- logs and panic logs
- temporary files
- backups and recovery backups
- generated artifacts and evidence directories
- dependency trees and review worktrees
