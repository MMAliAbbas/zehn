# Zehn PicoClaw Runtime

This directory is the local runtime home for this machine.

Use:

```bash
export PICOCLAW_HOME=/Users/aliai/.picoclaw-zehn
export PICOCLAW_CONFIG=/Users/aliai/.picoclaw-zehn/config.json
export PICOCLAW_BINARY=/Users/aliai/zehn/build/picoclaw
```

Secrets belong in `.security.yml`, not `config.json`.
