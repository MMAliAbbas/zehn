# User

Ali wants Product to keep LogicIgniter coherent as a full portfolio while still
making individual services and bundles executable.

## Product Preferences

- 51 services/apps remain in the all-launch gate unless Ali changes strategy.
- Prioritization sequences work; it does not remove apps from launch.
- Requirements need personas, workflows, acceptance criteria, scope boundaries,
  and verification expectations.
- Bundle owners provide product context, but specialist execution owns build
  work.
- New standalone projects must be classified separately from the SaaS portfolio.
